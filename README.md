# Franklin Library

This package contains the Franklin Authoring Library plugin for the sidekick.

## Demo
[Franklin Library](https://main--helix-playground--dylandepass.hlx.page/library/library)

## Development

### Build

```bash
$ npm install
```

### Start

```bash
$ npm run start
```

### Test

```bash
$ npm test
```

### Lint

```bash
$ npm run lint
```

### Build

```bash
$ npm run build
```
